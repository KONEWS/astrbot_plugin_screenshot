完美截图插件 (AstrBot Plugin)
这是一个为 AstrBot 开发的专属屏幕截图插件，致力于解决 Windows 系统下常见的截图缩放裁剪问题，并为多屏用户提供极致的截图体验。

🌟 核心优势 (优点)
突破多屏缩放限制：通过调用 Windows 底层更高级的 Per-Monitor DPI 接口 (SetProcessDpiAwareness(2))，完美解决 Win11/Win10 环境下多显示器混合缩放（如 125%、150%）时，截图发虚或只能截取四分之一画面的痛点。

双引擎极速响应与多屏支持：底层优先采用高性能 mss 库进行精准的多显示器画面抓取，当遇到极端环境时自动无缝降级至 Pillow，既保证了极低的延迟，又实现了超高的存活率。

全自动部署与自清洁机制：内置完整的 requirements.txt 依赖清单，支持 AstrBot 面板一键“即插即用”。更内置了异步文件自动清理器，每次截图发送完毕后会自动销毁硬盘原图，彻底杜绝服务器/电脑硬盘资源泄露。

极度自由的指令解析：内置多重路由智能剥离算法，彻底告别严格的空格限制（如 电脑截图2 或 电脑截图 2 5 均可完美识别），不会被框架误拦截触发 AI 聊天。

⚠️ 局限性说明 (缺点)
为了保持客观，安装前请注意本插件的以下局限性：

系统强绑定：由于大量调用了系统底层的画面渲染和 DPI API，本插件必须运行在 Windows 环境下。部署在纯命令行的 Linux 云服务器（无头环境）或 macOS 上将会报错并提示不支持。

物理显示要求：电脑不能处于锁屏状态，若是远程桌面连接，在最小化远程窗口时大概率会截出黑屏（此为 Windows 操作系统层面的物理安全限制）。

🚀 安装与使用
安装方法：

在本仓库下载打包好的 .zip 插件文件（或直接在本地将 main.py、metadata.yaml、requirements.txt 三个文件压缩为 zip）。

进入 AstrBot 的 WebUI 管理面板。

在插件管理页面选择“上传插件”，选中 .zip 文件并安装。

重启 AstrBot 后台（框架会自动在后台为你安装运行所需的依赖）。

使用指令：
在支持的聊天软件中对机器人发送（支持最高 4 个屏幕的快捷识别）：

电脑截图：立即截取主屏幕（屏幕 1）

电脑截图2：立即截取副屏幕（屏幕 2），无空格完美识别

电脑截图 2 5：将在 5 秒后，截取副屏幕（屏幕 2）

机器人收到后会回复倒计时提示，并在截图完成后发送全图。
