📸 AstrBot 完美截图插件 (v2.0.5)astrbot_plugin_screenshot 是一款专为 Windows 环境打造的 AstrBot 高稳定截图插件。经历了多个版本的迭代与底层重构，彻底解决了传统截图插件中常见的“指令呆板”、“多屏报错”、“资源泄露”以及“高分屏模糊”等痛点。

✨ 核心亮点 (优点)

🖥️ 完美适配 Win11 缩放 (DPI Aware)：引入了先进的 Per-Monitor DPI 适配机制。
运行 Win11 的高分辨率/高刷设备，能完美绕过系统的强制缩放，截出 1:1 清晰度的原图，彻底告别截图发虚、截不全的问题。

🧠 极度自由的指令解析：告别严格的空格限制。无论你是输入 电脑截图2 还是 电脑截图 2，框架底层的多重路由和智能字符剥离技术都能精准识别你的意图，不会再被误判为无效指令或触发 AI 聊天。

📺 多显示器与延迟支持：原生支持多屏环境。你可以自由指定截取主屏或副屏，并支持自定义延迟（最高兼容 4 个屏幕），方便你在截图前准备好需要展示的画面。

🛡️ 双重引擎容错机制：底层优先采用高性能的 mss 库进行多屏抓取；当遇到极端环境失效时，会自动无缝降级到原生 ImageGrab 引擎，极大保证了插件的存活率。

🧹 零资源泄露 (自清洁机制)：内建异步文件清理器。每次截图发送完毕后，后台任务会在 8 秒后自动销毁存储在硬盘上的原图，让机器人的服务器/电脑硬盘永远不会被截图垃圾塞满。

⚠️ 已知限制与注意事项 (缺点)系统强绑定：受限于底层的 API 调用，本插件仅支持 Windows 系统。在 Linux 或 macOS 等环境下使用会直接拦截并提示不支持。运行状态要求：由于 Windows 系统的安全机制，如果你的电脑处于锁屏状态、或者通过远程桌面连接但最小化了远程窗口，截图大概率会失败或截出一片黑屏（属于操作系统的物理限制）。依赖第三方库：本插件并非纯原生标准库编写，必须确保环境网络通畅以安装必要的依赖。

📦 安装指南将本插件文件夹（包含 .py 和 .yaml）放入 AstrBot 的 data/plugins/ 目录下。

💬 指令速查表只要包含前缀 电脑截图，你想怎么发就怎么发：用户输入法指令触发效果电脑截图立即截取 主屏幕 (屏幕 1)电脑截图2立即截取 副屏幕 (屏幕 2)，无缝兼容无空格输入电脑截图 2同上，带空格也完美识别电脑截图 2 5截取 副屏幕 (屏幕 2)，并在 5秒后 执行电脑截图2 5同上，前缀无空格，参数带空格，依然完美识别(注：最大支持快捷识别至 电脑截图4)
